# Sky-Lunna-Games
Olá.
Se você está lendo esta mensagem, é porque deseja jogar um pouco.
Seja muito bem-vindo(a) à Sky-Lunna-Games.
Por aqui, preparei um pequeno compilado de jogos desenvolvidos em linguagem C, utilizando o Visual Studio Code para compilação. Os jogos são simples, porém divertidos, e servem tanto para entretenimento quanto para testar habilidades.
Sinta-se à vontade para explorá-los da forma que preferir e jogar quantas vezes desejar.
Espero que tenha uma ótima experiência durante sua estadia. Caso tenha alguma dúvida, sugestão ou feedback, não hesite em informar.
Durante o processo de cada jogo ocorreu pequenos problemas de linguagen onde foi necessario a ajuda de assistentes virtuais
no caso Grox
Desejo uma excelente diversão!
